# libcpatch
Download, unpack, patch, and run ELF files with custom libc together with ld and debug symbols


### Usage:
`./libcpatch 2.23-0ubuntu11_amd64 ./chapter1`

### Supported libc versions:

2.23-0ubuntu10_i386
